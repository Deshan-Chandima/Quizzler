# Quizzler

Quizzler is an Android application that allows users to **create
quizzes**, **answer quizzes**, **download quizzes**, and **complete them
with a timer**. It is designed to be simple, fast, and easy to use.

## ‚úÖ What Quizzler Does

-   Create your own quizzes by adding questions and answers.
-   Take quizzes and receive instant scoring.
-   Download external quizzes and play them.
-   Timer-based quiz system to make attempts more challenging.
-   All quizzes are saved locally on the device.
